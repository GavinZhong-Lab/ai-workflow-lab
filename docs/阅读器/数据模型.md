# 阅读器模块 — 数据模型

## Prisma Schema 新增表

### Novel（小说）

```prisma
model Novel {
  id          String    @id @default(uuid())
  title       String
  author      String    @default("")
  description String    @default("")
  coverUrl    String    @default("")
  category    String    @default("")
  status      String    @default("SERIAL")   // SERIAL | COMPLETED
  wordCount   Int       @default(0)
  sourceType  String    @default("SCRAPED")  // SCRAPED | MANUAL
  sourceUrl   String?   @unique
  isActive    Boolean   @default(true)
  isFeatured  Boolean   @default(false)
  sortOrder   Int       @default(0)
  createdAt   DateTime  @default(now())
  updatedAt   DateTime  @updatedAt

  chapters    Chapter[]
  favorites   UserFavorite[]
  progress    UserReadingProgress[]
  scraperProgress ScraperNovelProgress[]

  @@index([category])
  @@index([isActive, isFeatured, sortOrder])
}
```

### Chapter（章节）

```prisma
model Chapter {
  id            String   @id @default(uuid())
  novelId       String
  chapterIndex  Int
  title         String
  content       String   @default("")
  wordCount     Int      @default(0)
  sourceUrl     String   @default("")
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt

  novel         Novel    @relation(fields: [novelId], references: [id], onDelete: Cascade)
  progress      UserReadingProgress[]

  @@unique([novelId, chapterIndex])
  @@index([novelId, chapterIndex])
}
```

### UserFavorite（收藏）

```prisma
model UserFavorite {
  id        String   @id @default(uuid())
  userId    String
  novelId   String
  createdAt DateTime @default(now())

  user      User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  novel     Novel    @relation(fields: [novelId], references: [id], onDelete: Cascade)

  @@unique([userId, novelId])
}
```

### UserReadingProgress（阅读进度）

```prisma
model UserReadingProgress {
  id         String   @id @default(uuid())
  userId     String
  novelId    String
  chapterId  String
  percentage Float    @default(0)
  createdAt  DateTime @default(now())
  updatedAt  DateTime @updatedAt

  user       User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  novel      Novel    @relation(fields: [novelId], references: [id], onDelete: Cascade)
  chapter    Chapter  @relation(fields: [chapterId], references: [id], onDelete: Cascade)

  @@unique([userId, novelId])
}
```

### ReaderBanner（横幅）

```prisma
model ReaderBanner {
  id        String   @id @default(uuid())
  title     String
  imageUrl  String
  linkUrl   String   @default("")
  sortOrder Int      @default(0)
  isActive  Boolean  @default(true)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
}
```

### ScraperLog（爬虫日志）

```prisma
model ScraperLog {
  id              String    @id @default(uuid())
  siteName        String
  status          String    // RUNNING | COMPLETED | FAILED
  message         String?
  durationHours   Int?
  novelCount      Int       @default(0)
  novelCompleted  Int       @default(0)
  novelIncomplete Int       @default(0)
  chapterCount    Int       @default(0)
  startedAt       DateTime?
  finishedAt      DateTime?
  createdAt       DateTime  @default(now())

  novels          ScraperNovelProgress[]
}
```

### ScraperNovelProgress（采集进度）

```prisma
model ScraperNovelProgress {
  id               String    @id @default(uuid())
  scraperLogId     String
  siteName         String
  novelTitle       String
  novelUrl         String
  author           String    @default("")
  novelId          String?
  status           String    // RUNNING | INCOMPLETE | COMPLETED | FAILED
  totalChapters    Int       @default(0)
  fetchedChapters  Int       @default(0)
  lastChapterIndex Int       @default(0)
  errorMessage     String?
  createdAt        DateTime  @default(now())
  updatedAt        DateTime  @updatedAt

  scraperLog       ScraperLog @relation(fields: [scraperLogId], references: [id])

  @@index([status])
  @@index([siteName, status])
}
```

## 表关系图

```
User
 ├── UserFavorite (1:N)
 │    └── Novel
 ├── UserReadingProgress (1:N)
 │    ├── Novel
 │    └── Chapter

Novel
 ├── Chapter (1:N)
 ├── UserFavorite (1:N)
 ├── UserReadingProgress (1:N)
 └── ScraperNovelProgress (1:N)

ScraperLog
 └── ScraperNovelProgress (1:N)

TranslationConfig (独立)
ReaderBanner (独立)
```

## 索引策略

- `Novel.category` — 分类过滤查询
- `Novel.isActive + isFeatured + sortOrder` — 列表排序
- `Novel.sourceUrl` — 爬虫去重（UNIQUE）
- `Chapter.novelId + chapterIndex` — 章节查询和排序（UNIQUE）
- `UserFavorite.userId + novelId` — 收藏去重（UNIQUE）
- `UserReadingProgress.userId + novelId` — 进度唯一（UNIQUE）
- `ScraperNovelProgress.status` — 断点续爬查询
- `ScraperNovelProgress.siteName + status` — 按站点筛选未完成
